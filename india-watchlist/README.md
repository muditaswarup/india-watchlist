# India Equity Watchlist

A personal equity research dashboard tracking Indian stocks across defence, pharma, power, fintech and consumption sectors.

**Live site:** https://YOUR-USERNAME.github.io/india-watchlist

## What's inside

- `index.html` — the full dashboard with live TradingView prices and charts
- Updated weekly with fresh prices, conviction scores, entry/exit levels and news

## How to update

1. Get the new `index.html` from Claude (say: *"weekly refresh — update the masterlist"*)
2. Replace `index.html` in this repo (drag and drop on GitHub.com or use the steps below)
3. GitHub Pages auto-deploys within 60 seconds — the live URL updates immediately

## Manual update via GitHub.com (no coding needed)

1. Go to your repository on github.com
2. Click on `index.html`
3. Click the pencil ✏️ edit icon (top right)
4. Select all → paste new content from Claude
5. Click **Commit changes** → **Commit directly to main**
6. Done — site updates in ~60 seconds
